""
Sign Assist - Installation Guide for Python Backend

PREREQUISITES:
1. Python 3.8 or higher installed
2. pip (Python package manager)

INSTALLATION STEPS:

Step 1: Install Python packages
Open Command Prompt (Windows) or Terminal (Mac/Linux) and run:

pip install flask flask-cors opencv-python mediapipe numpy pillow

Step 2: Verify installation
Run:
python --version
pip list

You should see flask, flask-cors, opencv-python, mediapipe, numpy, and pillow in the list

Step 3: Run the server
Navigate to your project folder and run:

python server.py

You should see:
"Sign Assist - AI Processing Server"
"Server starting on http://localhost:5000"

Step 4: Test the server
Open your browser and go to:
http://localhost:5000/api/health

You should see: {"status": "healthy", ...}

TROUBLESHOOTING:

Problem: "pip is not recognized"
Solution: Reinstall Python and check "Add Python to PATH"

Problem: "No module named flask"
Solution: Run: python -m pip install flask flask-cors

Problem: "opencv-python installation failed"
Solution: Try: pip install opencv-python-headless

Problem: Port 5000 already in use
Solution: Change port in server.py: app.run(port=5001)

OPTIONAL: Create virtual environment (recommended for production)

Windows:
python -m venv venv
venv\Scripts\activate
pip install flask flask-cors opencv-python mediapipe numpy pillow

Mac/Linux:
python3 -m venv venv
source venv/bin/activate
pip install flask flask-cors opencv-python mediapipe numpy pillow

RUNNING WITHOUT BACKEND:
The application works without the Python backend using browser-based AI.
The backend provides enhanced accuracy and features.

For questions or issues, check README.md
"""
